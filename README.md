# World Rank

세계 80억 인구 속에서 나의 위치를 확인할 수 있는 체험형 데이터 앱입니다.

## 주요 기능

- **라이프스타일 랭크**: 생활 수준 관련 질문에 답하면 전 세계 기준 백분위와 티어를 보여줍니다.
- **소득 랭크**: 연간 소득을 입력하면 전 세계 소득 분포에서의 위치를 계산합니다. PPP/MER 기준 선택 가능.
- **국가 크기 비교**: 인터랙티브 지도에서 국가 크기를 비교합니다.
- **글로벌 통계**: 다양한 글로벌 통계를 조회합니다.

## 사용 방법

### 라이프스타일 랭크
1. 질문에 차례대로 답변
2. 결과 화면에서 백분위/티어 확인

### 소득 랭크
1. 국가 선택 (통화 자동 설정)
2. 연간 소득 입력
3. PPP/MER 기준 선택
4. 결과 확인

## 지원 언어
한국어, 영어, 스페인어, 포르투갈어

## 데이터 출처
- 소득 분포: World Inequality Database (WID.world)
